あるベクトルとあるベクトルの類似度を調整する

ベクトルaがベクトルnと類似している（コサイン類似度が高い）という結果がでた場合
もう少し類似度を下げたいといった調整方法

通常ベクトル間の類似度は、ベクトルの内積で求める（ベクトルの正規化後）

Ca1 = Va * V1

これに重みとバイアスを加えることで調整可能とする

Ca1 = Va * (V1 * Wa + Ba)

ベクトルVaと、類似度を判定したいベクトルと調整後のコサイン類似度のn個の組合せから
最適なWa, Baを求める
